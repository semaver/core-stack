    Class ExecutableMember<T>Abstract

    abstract base class for invocable class members (constructors and methods); on top of ClassMember it holds the ordered collection of parameters and exposes parameter inspection (count, get by index) and parameter-decorator queries

    AbstractgetOwnParameterCount

    • getOwnParameterCount(): number

      method to get the parameter count declared directly on this member, read from the native function's arity via the descriptor api (unlike getKnownParameterCount, it does not investigate superclasses). Trailing rest parameters are not counted and parameters with a default value stop the count, e.g. myMethodA(...args) returns 0, myMethodA(someArg1, someArg2, ...args) returns 2, myMethodA(someArg1, someArg2 = 1, ...args) returns 1

      Returns number

      own parameter count declared directly on this member
